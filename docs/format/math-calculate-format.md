# ➗Math Calculate Format

{% hint style="warning" %}
You need set `math.enabled` option to `true` in `config.yml` file and then use format like `%totem_center_y%+5` instead.
{% endhint %}

The **Math Calculation Format** provided by **MythicTotem** are almost identical to those in UltimateShop. Therefore, they will not be elaborated on in this wiki. Please refer to UltimateShop's wiki for a detailed introduction about them. Click [here](https://ultimateshop.superiormc.cn/format/math-calculate-format) to view.
