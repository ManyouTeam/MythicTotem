# ⌨️Commands & Permissions

## Commands with Permission

#### /mythictotem help

View the help info of this plugin.

#### /mythictotem list

View the list of valid totems. Require `mythictotem.admin` permission to use.

#### /mythictotem reload

Reload the plugin. Require `mythictotem.admin` permission to use.

## Permissions

mythictotem.bypass.protection - Bypass protection check for totem active.
